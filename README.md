# BERTopic The App

This Streamlit application provides an interactive interface for topic modeling using the BERTopic library. It allows users to analyze free text data, either by manual input or by uploading a CSV file, and visualize the resulting topics.

## Features

-   **Interactive Topic Modeling**: Leverage the power of BERTopic to discover topics in your text data.
-   **Flexible Data Input**:
    -   **Manual Input**: Paste text directly into a text area, with each line treated as a document.
    -   **CSV Upload**: Upload a CSV file and select the column containing the text you want to analyze.
-   **Customizable Model Parameters**: Fine-tune the topic modeling process through a user-friendly sidebar interface:
    -   **UMAP Parameters**: Adjust `n_neighbors`, `n_components`, `min_dist`, and the distance `metric`.
    -   **BERTopic Parameters**: Configure the `Minimum Topic Size` and select the `Language` for stopword removal.
-   **Rich Visualizations**: Explore the discovered topics with a variety of interactive plots generated by BERTopic:
    -   Topic Frequency Bar Chart
    -   Topic Hierarchy Dendrogram
    -   Topic Similarity Heatmap
    -   Top Words per Topic
-   **Control Visualizations**: A simple checkbox to show or hide all the detailed visualizations.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd bertopic-app
    ```

2.  **Create a virtual environment (recommended):**
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
    ```

3.  **Install the dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1.  **Run the Streamlit application:**
    ```bash
    streamlit run streamlit_app.py
    ```

2.  **Open your browser**: Navigate to the local URL provided by Streamlit (usually `http://localhost:8501`).

3.  **Configure and Analyze**:
    -   Use the sidebar to set your desired UMAP and BERTopic parameters.
    -   Choose either the "Manual Input" or "CSV Upload" tab to provide your data.
    -   Click the **Start Analysis** button to begin the topic modeling process.
    -   View the results and interactive visualizations that appear.
